# 🚀 ChainTrackr – Crypto Portfolio Tracker

ChainTrackr is a modern iOS application built using SwiftUI that enables users to track cryptocurrency prices, monitor market trends, and explore digital assets through a clean and intuitive interface.

---

## 📱 Features

- 📊 Live Crypto Tracking  
  View real-time cryptocurrency prices and market data.

- 📈 Interactive Charts  
  Analyse trends with smooth and visually appealing charts.

- 🔍 Search & Explore  
  Quickly find coins and explore detailed insights.

- 💡 Modern UI/UX  
  Designed with SwiftUI for a seamless and responsive experience.

- ⚡ Fast & Lightweight  
  Optimised for performance with efficient data handling.

---

## 🛠 Tech Stack

- SwiftUI – UI development  
- URLSession / API Integration – Fetch live crypto data  
- Combine – Reactive data handling
